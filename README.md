# 🚀 LearnEx – Personalized Learning Platform

LearnEx is an interactive learning platform designed to provide personalized educational experiences with AI chatbot integration and secure user authentication.

Built using modern frontend technologies and Supabase as a Backend-as-a-Service (BaaS).

---

## 📌 Features

- 🔐 Secure User Authentication (Supabase Auth)
- 📝 Notes Management System
- 🤖 AI Chatbot Integration
- 📊 Real-time Database Updates
- 🎯 Clean and Responsive UI
- ☁️ Serverless Backend using Supabase

---

## 🛠 Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend (BaaS)
- Supabase
- PostgreSQL

### Tools & Environment
- Node.js (for local development)
- Git & GitHub
