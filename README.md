Personalized Learning Chatbot (Gemini + Node.js Proxy + Supabase) 🧠
This project is a personalized learning application featuring an AI-powered tutor built using the Google Gemini API. It uses a secure Node.js proxy server to manage the API key and handles user data, progress tracking, and chat history using Supabase.

✨ Features
Personalized Tutoring: The Gemini model is given specific System Instructions and the user's learning history to provide tailored, encouraging, and context-aware guidance.

Secure API Handling: A dedicated Node.js (hf-proxy.js) server securely manages the Gemini API key, preventing exposure in the client-side code.

Supabase Integration: Used for user authentication, storing chat history, and managing user progress data.

Modern Web Stack: Built with modern JavaScript, HTML, and CSS (likely scaffolded with Vite).
